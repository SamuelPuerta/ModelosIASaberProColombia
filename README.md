# **PROYECTO KAGGLE - Pruebas Saber Pro Colombia**

**Entrega 1 :** Exploración de tablas de datos sobre el rendimiento de los estudiantes en las pruebas Saber Pro.

**Entrega 2 :** Preprocesado y limpieza de la tabla de datos train.csv sobre el rendimiento de los estudiantes en las pruebas Saber Pro.

**Entrega Final :** Se implementa un modelo de clasificación para predecir el rendimiento global de estudiantes en las Pruebas Saber Pro de Colombia, utilizando XGBoost con optimización de hiperparámetros mediante Optuna y pseudo-etiquetado.

El contexto del proyecto es predecir el desempeño de estudiantes (bajo, medio-bajo, medio-alto, alto) basado en datos socioeconómicos, institucionales y estadísticos.
